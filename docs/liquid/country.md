# country.name

The full name of the country.

# country.id

The internal Id of the country.

# country.code

Returns a two-letter [ISO 3166](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2) code of the country.

# country.continent

Returns the two letter code of the continent.

Can be either:

| Code | Continent |
|----|---------|
| AN | Antartica |
| OC | Oceania |
| AS | Asia |
| NA | North America |
| SA | South America |
| EU | Europe |
