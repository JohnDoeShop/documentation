currency.symbol

Returns the currency symbol.

Example

{{ currency.symbol }}
<!-- Outputs: $ -->

currency.id

Returns the currency internal Id.

currency.code

Example

{{ currency.code }}
<!-- Outputs: USD -->

Returns a three-letters ISO 4217 currency code.

currency.name

Returns the full currency name.

Example

{{ currency.name }}
<!-- Outputs: Euro -->
